# Intent-Aware Multi-Stage Defense for Personalized Recommender Systems

This repository contains the code, experiment scripts, and thesis artifacts for an intent-aware multi-stage defense prototype for recommender systems.

## Repository structure
- `scripts/`: runnable experiment scripts
- `src/`: modular reusable code
- `configs/`: configuration files
- `results/tables/`: main result tables
- `results/plots/`: thesis figures
- `chapters/`: LaTeX thesis chapters

## Data
This repository does not redistribute all raw datasets directly.
The project uses:
- MIND-small
- Aegis 2.0
- a small moral_education seed pool

Please follow the thesis appendix and project documentation to reconstruct the processed environment.

## Main outputs
The repository includes:
- code for preprocessing, training, reranking, and evaluation
- thesis-ready tables and plots
- appendix-ready summary artifacts

## Reproducibility
Create the environment, prepare the processed data, and run the scripts in `scripts/` in the documented order.
